# Strava Tracker

The Strava Tracker application is designed to track activities of a specific Strava Club. It provides functionality to store information about activities and enables users to compare statistics between athletes based on date or type of activity.

## Backend

The backend is a Java Spring Boot application that fetches data from the Strava API about athletes and activities. It stores the data in a PostgreSQL database and provides endpoints for the frontend.
